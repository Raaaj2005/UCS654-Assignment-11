# 📊 Parameter Estimation

## 📖 Overview
This repository contains the PDF document detailing solutions for parameter estimation problems, specifically focusing on mathematical statistics and probability distributions.

## 🎯 Topics Covered
* Maximum Likelihood Estimates (MLE) for the mean and variance of a Normal Population.
* Minimum Mean Square Error (MMSE) Estimation for a Binomial distribution parameter using a Bayesian approach.

## 🚀 Usage
Download or view the uploaded PDF file to access the step-by-step mathematical derivations.
